### Part 1：Hook（0:00-0:30）

**画面：** 猿哥出镜，语速快

**猿哥：**
> 三个月前，我做了一个决定：让 AI 替我做我每天最繁琐的工作。
> 我花了一个周末，搭了套全自动流水线。
> 然后我观察了 30 天。
> 今天聊一个话题：AI 到底能不能替代一个内容创作者？
> 答案是……能，也不能。

**画面：** 黑屏白字弹出标题

> 我用AI搭了个GitHub Trending日报机器人
> 连续更新30天后我慌了

---

### Part 2：为什么要做这个（0:30-1:30）

**画面：** 猿哥出镜 + GitHub Trending 页面截图

**猿哥：**
> 我是猿来AI，每天做的事：刷 GitHub Trending，挑 5 个 AI 项目，写分析，更新网站。
> 听起来不难？但每天都要做。
> 选题、读 README、查资料、写分析、更新数据、发文章。
> 重复劳动占了我 60% 的时间。
> 我就想：这些能不能让 AI 干了？

**画面：** 人工 vs 自动化流程对比动画

**猿哥：**
> 于是我开始搭这套流水线。

---

### Part 3：动手搭建（1:30-4:00）

**画面：** 终端 + VS Code 操作实录

**猿哥（画外音）：**
> 第一步：爬数据。
> GitHub Trending 没有官方 API，用 Headless Browser 定时抓取。

**画面：** 代码展示

```python
# 核心代码
browser.navigate("github.com/trending")
projects = extract_all_projects()
```

**猿哥（画外音）：**
> 第二步：让 AI 分析。
> 把项目标题、描述、Star 数喂给大模型。
> 我用的不是「帮我写分析」这种模糊指令。
> 而是精确告诉它：读者是 AI 开发者，关注实用性和趋势判断，短句，每段不超 3 行。

**画面：** ❌「帮我写篇分析」 vs ✅ 精确 prompt 对比

**猿哥（画外音）：**
> 第三步：自动化发布。
> JSON → 更新网站 → 发到公众号草稿箱。
> 一条命令搞定。

**画面：** 终端执行部署脚本

---

### Part 4：反转来了（4:00-5:30）

**画面：** 节奏放缓，猿哥出镜

**猿哥：**
> 头一周，效果惊艳。
> 每天自动出 5 篇分析，数据准确、格式工整、按时发布。
> 我心想：成了。以后躺着就行。

**画面：** 数据飙升动画

**猿哥：**
> 但两周后我发现一个问题。
> 同一个类型反复出现——Agent 框架、Token 压缩、记忆引擎。
> AI 判断「热门」的标准很单一：Star 数涨得快 = 好项目。
> 但读者真的每天都要看 Agent 框架吗？

**画面：** AI 选出的项目列表（全 Agent 相关） vs 猿哥手动选的（混入冷门有趣项目）

**猿哥：**
> AI 会选「安全」的爆款。
> 我会选「有趣」的项目——哪怕它只有 500 星。
> 差的那一点，叫「品味」。

---

### Part 5：技术架构（5:30-6:30）

**画面：** 流水线架构图（打开 pipeline-architecture.html）

**猿哥（画外音）：**
> 这是我的全套流水线，全部开源。

```
GitHub Trending → Headless → AI 分析 → GitHub Pages + 公众号 API
```

| 环节 | 工具 |
|------|------|
| 数据爬取 | Headless Browser |
| AI 分析 | DeepSeek / Claude |
| 网站托管 | GitHub Pages |
| 自动发布 | WeChat API |

---

### Part 6：猿哥观点 & CTA（6:30-8:00）

**画面：** 猿哥出镜

**猿哥：**
> 30 天实验结束了。结论很简单：
>
> **AI 能干 80% 的活。但剩下的 20% 是灵魂。**
>
> 它可以收集信息、整理数据、生成初稿。
> 但它不知道什么项目「有意思」。
> 它不知道你的读者今天想看什么。
>
> 所以我现在的工作流：
> AI 干粗活——刷数据、写初稿、排版发布。
> 我干细活——选题、定调、加观点。
>
> 以前每天花 2 小时做日报。
> 现在花 20 分钟。
> 省下的时间，用来想更好的选题。

**画面：** 文字浮现

```
AI 替你干活
你替读者活着
```

**猿哥：**
> 这就是我在 Build in Bilibili 想说的。
> 不是用 AI 替代人，是用 AI 放大人的价值。
>
> 我是猿来AI，关注我，每天给你扒一个 GitHub 热门 AI 项目。
> 下期见 🐒
